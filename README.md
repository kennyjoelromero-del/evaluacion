TÍTULO DEL PROYECTO  
Asistente de Finanzas Personales Básico

Descripción del Proyecto  
Este proyecto consiste en un asistente financiero sencillo que permite registrar ingresos, registrar gastos, calcular el balance disponible y mostrar un resumen financiero.
El sistema funciona por consola y utiliza estructuras lógicas, funciones, validaciones y diagramas de flujo para representar su funcionamiento.
Forma parte de la evaluación final de la asignatura, integrando análisis, diseño, programación estructurada y control de versiones con GitHub.

Objetivo del Sistema  
Desarrollar una herramienta simple y funcional que permita al usuario llevar un control básico de sus finanzas personales mediante el registro de ingresos, gastos y el cálculo del balance.

Funcionalidades Principales

Registrar ingresos válidos (mayores a 0)

Registrar gastos válidos (mayores a 0)

Calcular el balance total (ingresos menos gastos)

Mostrar un resumen financiero

Validaciones de entrada

Menú interactivo con ciclo repetitivo

Estructura modular basada en funciones

Estructura del Repositorio  
Carpeta src: contiene el archivo main.py
Carpeta diagramas: contiene los diagramas del sistema
Carpeta documentacion: contiene el documento final en PDF
Archivo README.md: contiene la descripción del proyecto

Tecnologías Utilizadas  
Python 3
RAPTOR (diagramas de flujo)
Git y GitHub
Programación estructurada

Cómo Ejecutar el Programa

Instalar Python 3

Descargar o clonar el repositorio

Entrar a la carpeta src

Ejecutar el archivo main.py con el comando: python main.py
